<identifier>:
First character is a letter, any proceeding characters are letters or numbers.

<type>
A primitive type: string, integer, boolean, etc.

<value>:
Either an identifier (for a variable) or a literal (1, "Hello", true)

<condition>:
Either equals, notequals, greaterthan, lessthan

Class Declaration:
`class <identifier>`

Method Declaration:
`method <identifier = name> (<type> <identifier = name>...) [returns <type>] end <identifier = name>`

Variable Declaration:
`var <type> <identifier = name> [= <value>]`

Print Statement:
`print <value>`

Get input:
`getinput <variable>`

Run Method:
`run <method name>`

If/Else Statement:
`if <condition> <variable a> <variable b> ... end`

While Loop:
`while <condition> <variable a> <variable b> ... end`
